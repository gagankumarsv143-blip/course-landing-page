# Course Landing Page

This repository contains a professional, responsive course landing page built using HTML and CSS only.

## Features

- **Header** with course title and tagline that animates on load.
- **About section** with short description.
- **Features section** showcasing what students will learn, using cards with 3D hover effects.
- **Benefits section** outlining reasons to join.
- **Call-to-action** area with an animated "Enroll Now" button.
- **Contact section** containing a simple form.
- Fully **mobile responsive** with media queries.
- **Dark theme** with black backgrounds and white text throughout.
- **CSS animations**, **3D transforms** and floating shapes both in header and page-wide for a professional, immersive look.

## How to view

1. Open `index.html` in a browser (double-click or use `Live Server` extension).
2. Resize the browser window or open on a mobile device to see responsive behavior.

Feel free to customize the text, colors, and add images or videos as needed.
